Make Your Own Quiz

  1. Type a quiz title and description.
  2. Click "Add New Question".
  3. Type the question text, using <br> to add line breaks.
  4. Click "Add New Answer" and add at least two answers per question.
  5. Select the correct answer using the radio button.
  6. Add as many questions as you'd like, then click "Submit Quiz."
